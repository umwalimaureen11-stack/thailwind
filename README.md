# 🚀 Tailwind CSS Hero Section – Frontend Assignment

## Project Overview
This project is a modern and responsive **Hero Section** built using **HTML5 and Tailwind CSS**.  
It replicates a professional SaaS / mobile application landing page design with focus on layout, spacing, and typography.



##  Technologies Used
- HTML  
- Tailwind CSS (CDN)

---

##  Features
- Strong headline section  
- Supporting description text  
- Call-to-action (CTA) buttons  
- App/phone mockup image  
- Fully responsive design  
- Clean and modern layout  

---

##  Responsiveness
The design adapts to:
- Mobile devices  
- Tablets  
- Desktop screens
